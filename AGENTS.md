# Kanta — agent skills

Visual layout / settings editor for Kanata. Go + Wails UI. Cross-platform (all platforms Kanata targets).

## Agent skills

### Issue tracker

GitHub Issues on `sjclayton/kanta` via the `gh` CLI. External PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default five-role vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at repo root, created lazily. See `docs/agents/domain.md`.
